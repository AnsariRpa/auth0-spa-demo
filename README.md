# Auth0 SPA Login Demo

## Description
This project demonstrates a simple login flow using Auth0 in a Single Page Application (SPA).  
It uses the Auth0 SPA SDK to authenticate users and retrieve profile information without building a custom authentication system.

---

## Features
- Login with Auth0
- Logout functionality
- Displays authenticated user profile data
- Secure authentication using OpenID Connect (OIDC)

---

## Technologies Used
- HTML
- JavaScript
- Auth0 SPA SDK
